# Explainable AI-Based Loan Approval Prediction System

## Project Overview

This project predicts whether a loan application should be approved using Machine Learning. It integrates Explainable AI (SHAP) to provide transparent explanations for each prediction, helping improve trust in automated credit decision-making.

---

## Features

- Loan approval prediction
- Multiple Machine Learning models
- Explainable AI using SHAP
- Feature importance analysis
- User-friendly interface
- Performance evaluation

---

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- Matplotlib
- Streamlit
- Jupyter Notebook

---

## Dataset

- Loan Approval Dataset (Kaggle)

---

## Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost
- K-Nearest Neighbors

---

## Results

- Random Forest and XGBoost achieved the highest prediction accuracy.
- SHAP identified CIBIL score, income, and loan amount as the most influential features.
